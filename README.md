# 🚀 AI-Powered Car Price Analysis Dashboard

## 📌 Project Overview

The **AI-Powered Car Price Analysis Dashboard** is an interactive data visualization project built using **Streamlit, Pandas, and Plotly**. It allows users to explore and analyze car dataset insights such as pricing, fuel type, mileage, and more.

This project demonstrates practical implementation of **data analysis, visualization, and dashboard development** skills.

---

## 🎯 Features

* 📊 Interactive data visualization using Plotly
* 📁 CSV dataset integration
* 🧹 Data cleaning and preprocessing
* 📈 Price analysis by car model
* ⚡ Real-time dashboard updates

---

## 🛠️ Technologies Used

* Python 🐍
* Streamlit
* Pandas
* Plotly

## ▶️ How to Run the Project

### Step 1: Install dependencies

```bash
pip install streamlit pandas plotly
```

### Step 2: Run the app

```bash
streamlit run app.py
```

### Step 3: Open in browser

```
http://localhost:8501
```
